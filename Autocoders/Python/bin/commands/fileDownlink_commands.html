Command: fileDownlink:FileDownlink

Mnemonic ID Description Arguments
Argument Type Description
FileDownlink_SendFile 501 (0x1f5) Read a named file off the disk. Divide it into packets and send the packets for transmission to the ground.
sourceFileName Fw::CmdStringArg The name of the on-board file to send
destFileName Fw::CmdStringArg The name of the destination file on the ground
FileDownlink_Cancel 502 (0x1f6) Cancel the downlink in progress, if any
