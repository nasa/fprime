Svc::Poly Port

Svc::Poly Port

1. Introduction

The Svc::Poly is used to pass the polymorphic Fw::PolyType between components. See the Svc::PolyType description for details on the class.

The arguments are as follows:

Argument|Description|Values -----—|--------—|---— entry | The index of the entry in the table status | The status of the measurement | MEASUREMENT_OK = normal measurement, MEASUREMENT_FAILURE = error reading measurement, MEASUREMENT_STALE = measurement out of date time | The time tag of the measurement val | The value of the measurement
