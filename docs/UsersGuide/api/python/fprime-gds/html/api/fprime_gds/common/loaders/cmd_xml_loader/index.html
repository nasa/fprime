Constructs and returns python dictionaries keyed on id and name

This function should not be called directly, instead, use get_id_dict(path) and get_name_dict(path)

Parameters

path – Path to the xml dictionary file containing command information

Returns

(id_dict, name_dict). The keys are the events’ id and name fields respectively and the values are CmdTemplate objects

Return type

A tuple with two command dictionaries (python type dict)
