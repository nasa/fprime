fprime_gds.common.adapters.ip¶

ip.py:

Module containing a comm-layer adapter for the Tcp/UDP/IP stack. This pairs with the F prime component “SocketIpDriver” in order to read data being sent via Tcp or Udp. This is the default adapter used by the system to handle data sent across a Tcp and/or UDP network interface.

@author lestarch

class fprime_gds.common.adapters.ip.IpAdapter(address, port)[source]¶

Bases: fprime_gds.common.adapters.base.BaseAdapter

Adapts IP traffic for use with the GDS ground system. This serves two different “servers” both on the same address and port, but one uses TCP and the other uses UDP. Writes go to the TCP connection, and reads request data from both. This data is concatenated and returned up the stack for processing.
