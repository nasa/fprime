fprime_gds.common.testing_fw.predicates¶

predicates.py:

This file contains basic predicates as well as event and telemetry predicates used by the gds_test_api.py. The predicates are organized by type and can be used to search histories.

koran

The event_predicate checks that the telemetry object is an instance of EventData and will raise an error if the check fails. Then event_predicate will evaluate whether event’s EventData fields satisfy the id_pred, value_pred, and time_pred specified.
